# 😂 React Joke App

A simple React app that fetches random jokes from an API.

## 🚀 Features
- Get random jokes with one click
- Loading state while fetching data
- Clean and simple UI

## 🛠️ Tech Stack
- React (Vite)
- TypeScript
- CSS

## 🌐 API Used
https://official-joke-api.appspot.com

## ▶️ How to Run Locally

```bash
npm install
npm run dev
